🚀 Como publicar no GitHub Pages

O que você vai precisar

Uma conta gratuita no github.com. O GitHub Pages hospeda arquivos estáticos (HTML, CSS, JS) de graça, sem servidor.

Passo a passo

  1. Crie um repositório: Acesse github.com/new, dê um nome (ex: consolidador-nf), marque Public e clique em Create repository.
  2. Faça upload do arquivo: Na página do repositório, clique em Add file → Upload files, arraste o arquivo consolidador_nf_boletos.html e confirme com Commit changes.
  3. Renomeie para index.html (opcional): Se quiser um link mais limpo, clique no arquivo → ícone de lápis → renomeie para index.html → Commit changes.
  4. Ative o GitHub Pages: Vá em Settings → Pages (menu lateral). Em Source, selecione Deploy from a branch, branch main, pasta / (root). Clique em Save.
  5. Aguarde ~1 minuto: O GitHub vai publicar automaticamente. O link aparece no topo da mesma página: https://seu-usuario.github.io/consolidador-nf/
  6. Compartilhe com o time: Copie e envie esse link. Qualquer pessoa com o link pode acessar direto no navegador, sem instalar nada.
💡 Atualizar o sistema: Basta fazer upload de uma nova versão do arquivo HTML pelo mesmo caminho (Add file → Upload files). O GitHub Pages atualiza automaticamente em ~1 minuto.

Configurar o Google Sheets (opcional)

Para sincronizar o histórico com o Google Sheets, crie um Apps Script seguindo estes passos:

  1. Abra o Google Sheets e crie uma planilha chamada Historico NF.
  2. No menu, clique em Extensões → Apps Script.
  3. Apague o código existente e cole o código abaixo:
function doPost(e) { try { const data = JSON.parse(e.postData.contents); const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName('Historico NF'); if (!sheet) { sheet = ss.insertSheet('Historico NF'); sheet.appendRow(['Nº NF','Data Emissão','Remetente','Destinatário','Valor','Importado em']); } const rows = Array.isArray(data) ? data : [data]; rows.forEach(r => sheet.appendRow([ r.nf, r.dataEmissao, r.remetente, r.destinatario, r.valor, r.importadoEm ])); return ContentService .createTextOutput(JSON.stringify({ok:true,added:rows.length})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ok:false,error:err.message})) .setMimeType(ContentService.MimeType.JSON); } } function doGet(e) { return ContentService .createTextOutput(JSON.stringify({ok:true,status:'online'})) .setMimeType(ContentService.MimeType.JSON); }
  1. Clique em Implantar → Nova implantação → tipo App da Web.
  2. Em Quem tem acesso, selecione Qualquer pessoa. Clique em Implantar e autorize.
  3. Copie a URL gerada (começa com https://script.google.com/macros/s/...).
  4. No consolidador, clique em 📊 Sheets, cole a URL e clique em Salvar URL.
⚠️ A partir daí, todo novo lote processado é enviado automaticamente para a planilha.
